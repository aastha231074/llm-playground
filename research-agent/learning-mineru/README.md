# MinerU Learning Path

Welcome to the comprehensive learning guide for **MinerU** - an enterprise-grade PDF-to-Markdown conversion tool optimized for AI/LLM workflows.

## What is MinerU?

MinerU is an open-source tool that transforms complex PDF documents (scientific papers, reports, textbooks) into clean, machine-readable formats like Markdown and JSON. It excels at preserving document structure while extracting text, images, tables, and mathematical formulas.

## Who is this guide for?

- Developers building RAG (Retrieval-Augmented Generation) systems
- Data scientists preparing document datasets for LLM training
- Researchers processing academic papers and scientific literature
- Anyone who needs to extract structured content from PDFs at scale

## How to use this learning path

This guide is structured in 5 progressive levels:

1. **Level 1: Overview & Motivation** - Understand what MinerU solves and why it exists
2. **Level 2: Installation & Hello World** - Get MinerU running with your first conversion
3. **Level 3: Core Concepts** - Learn the fundamental architecture and processing modes
4. **Level 4: Practical Patterns** - Build real-world solutions with common use cases
5. **Level 5: Next Steps** - Discover advanced topics and resources

Each level builds on the previous one. Work through them sequentially for the best learning experience.

- Basic command-line familiarity
